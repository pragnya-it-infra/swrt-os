SWRT-OS

SWRT-OS In order to enhance efficiency and resource usage, unnecessary modules and services were previously removed during the development phase of SWRT-OS, which is based on Debian-11 Bullseye and the most recent kernel version. totally customized lightweight icewm desktop with a unique tool menu. 500 preinstalled tools, divided into 20 different categories. 

We are newly integrate with chatGPT in terminal and GPT3. The project's goal is to use artificial intelligence to fully load scanning, vulnerability scanning/analysis, and exploitation. This design strategy also made it possible to carefully consider each package and component that would be SWRT-OS included in the distribution. It inspired the OS to create a user-friendly setting. 
With all the available OS around, there is always a process of installation of tools required for Bug bounty. Will 400+ tools available in the market, it is always tough to choose on what to install and what not. There is always multiple dependencies required for certain installations. To overcome all of those, this OS will be a one-step solution. It covers the tools installations and performance coverage.
SWRT-OS ISO contains a full live system allowing you to try it before you install. The live system contains an ICEWM desktop and software suite.

We"re planning to do the first official release of SWRT-OS Linux sometime in the next fews days which will include the same debian anywhere installer everyone knows and loves, and also some other big and exciting changes which have been in the works to make SWRT-OS Linux a distro in its own right.
Positive developments are taking place, and the new distribution has a promising future. For information on the next release, check back soon.
