SWRT-OS

The development of SWRT-OS, which is based on Debian 11 Bullseye and the most recent kernel version by removing irrelevant modules and services in order to improve efficiency and resource utilisation.
Completely personalised, lightweight icewm desktop with a unique tool menu.
200 built-in tools, split into 20 different categories.
We have recently integrated with GPT3 and chatGPT in the terminal.
The project's objective is to fully execute load scanning, vulnerability scanning and analysis, and exploitation using artificial intelligence.
The thorough consideration of each package and component that would be included in the distribution of SWRT-OS was indeed made possible by this design concept.
It prompted the OS to design a user-friendly environment.
There is usually a method of installing the tools required for bug bounty with all the OS choices available. With 400+ tools on the market, it might be difficult to decide which ones to install. For certain setups, numerous dependencies are always needed.
This OS will be a one-step solution to all of those issues. It includes coverage for tool installations and performance.
SWRT-OS ISO contains a full live system allowing you to try it before you install. The live system contains an ICEWM desktop and software suite.
The first official release of SWRT-OS Linux is scheduled to occur within the next few days and will include the well-known and renowned debian anywhere installer in addition to several other exciting new features.
